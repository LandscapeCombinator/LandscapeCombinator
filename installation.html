InstallationÔÉÅ

  1. Download the plugin from Unreal Engine Marketplace. For personal projects, you may use the GitHub sources.

  2. Add the plugin to Unreal Engine.

  3. Open a new project and enable the Landscape Combinator plugin.

  4. In the Content Browser, make sure that Show Plugin Content, Show Engine Content, and Show C++ Classes are enabled.

Note

If you plan to use Viewfinder Panoramas or Litto 3D Guadeloupe heightmaps, please install 7-Zip and add it to your Windows PATH environment variable as follows. Press the Windows button on your keyboard or on the bottom left of your screen, and write PATH with your keyboard. Click on Edit the system environment variables, then press the Environment Variables button. Click on PATH in the System Variables list, and press the Edit button at the bottom. Click on New, and paste the path to your 7-Zip installation (which is typically C:\Program Files\7-Zip).
